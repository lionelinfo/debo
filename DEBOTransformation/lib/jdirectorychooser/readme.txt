Description:
============

The JDirectoryChooser provides user-friendly GUI for manipulating directories 
from Java application. User can either simply choose an existing directory 
from a tree or edit file system directories' tree structure by creating 
new directories, renaming, copying, moving or deleting existing directories.



System Requirements:
====================

Java Runtime Environment 1.4 or later. 

The JDirectoryChooser is a 100% Pure Java and has no external dependencies.
